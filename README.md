# DSA Progress Tracker

A Java-based console application designed to help students track their Data Structures and Algorithms practice.

-> Features
- Add DSA questions
- Track solved problems
- Monitor progress
- Menu-driven interface
- Built using Java and OOP concepts

-> Technologies Used
- Java
- OOP
- ArrayList
- Git & GitHub

-> Future Improvements
- File storage
- Search and filter questions
- Progress statistics
- GUI version
